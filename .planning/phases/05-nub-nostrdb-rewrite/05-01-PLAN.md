    - `nostrdb.query` / `nostrdb.query.result` — filter-based event retrieval

    - `nostrdb.count` / `nostrdb.count.result` — count matching events
    - `nostrdb.subscribe` / `nostrdb.sub.event` / `nostrdb.sub.eose` — live subscription with push delivery
    - `nostrdb.unsubscribe` — close a live subscription
